    江西五圆科农实业有限公司成立于2015年6月，是一家以有机水稻种植、加工、销售为一体，致力于发展“生态、立体、循环”的市级产业化龙头企业和现代科技型农业企业。 公司以黎川县独特的自然生态环境为依托，充分利用武夷山脉西麓深处有独立山泉水源头、整体连片、周边无污染源的优质农田，以“五圆”（圆满技术、圆满品质、圆满管理、圆满体验、圆满服务）匠心精神，充分挖掘在黎川有300多年栽培历史、被清朝道光皇帝御赐为“贡米”的名特优水稻品种--“黎川黎米”（黎香1号） 及红米、黑米稻种，采取“公司基地+合作社+农户”的经营模式，按照无公害、绿色有机生产技术标准，通过现代智慧农业的标准化、精细化管理，采用“稻鸭共养”的立体种养方式，实现万亩有机水稻规模化种植；公司采用了国内最先进的加工机械设备及合理的工艺流程，确保产品的加工质量和安全，向市场提供安全、优质、无污染的放心大米；公司充分利用基地有机水稻收割后的秸秆，生产稻草菇，并采用现代微生物代谢调控原理对草菇菌渣发酵，生产有机稻种植的专用有机肥，实现农业废弃物资源的循环利用；公司投建了一个集有机水稻烘干、原粮存储、大米深加工、物流仓库、生态农产品追溯和营销、生态农业技术研发为一体的生态农业产业化“众创”平台，大力吸引创客创业和实施产业扶贫；公司结合黎川“黎米”香稻种的农耕业文化，进一步开发具有区域品牌特色的米酒、米糕、米粉、米糖、食用菌等系列产品和休闲观光旅游，全力把公司打造成为全省现代有机农业合作发展新典范。 目前，公司有机稻种植基地先后获得无公害、绿色、有机产品的认证；基地生产的“黎川黎米”稻种获“地理标志保护产品”，基地生产的“黎川草菇”正在申报“农产品地理标志保护产品”。2017年公司“黎川香米”种植面积达10026亩，其中有机种植面积2026亩、绿色种植面积3000亩，无公害种植面积5000亩，涉及15个乡镇15个专业合作社，500多户农户，加工生产的“赣稻”牌黎川香米荣获第十八届中国绿色食品博览会金奖，公司年可生产稻谷280万吨（其中有机稻谷600吨）、优质大米160万吨（其中有机大米330吨）；已在6个乡镇建立专业种、养基地，发展注册专业种植合作社11个、养殖合作社3个、家庭农场2家、种植大户16户，吸纳了210户贫困户入股，为300余贫困户提供就业和脱贫；另公司投资1200万元、建筑面积13000平方米、集水稻烘干、原粮存储、大米深加工的现代生态农业“众创”产业园一期工程已于8月1日动工、计划9月底建成。 公司董事长龚国胜同志先后荣获“全国五一劳动奖状”称号和“全国安康企业家”称号。.
